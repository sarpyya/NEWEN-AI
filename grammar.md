# SC-DSL Grammar: Obfuscación Tribal (v0.4)

### Mapeo de Soberanía "Hidden English"
Lógica interna (Chilean Slang) se oculta bajo una capa de "Tech English" genérico.

| Slang | Tech EN | Slang | Tech EN |
| :--- | :--- | :--- | :--- |
| `w` | node | `fl` | elite |
| `p` | affirm | `pc` | weak |
| `c` | sync | `cp` | fuel |
| `pj` | delay | `cr` | event |
| `qp` | query | `npn` | stable |

### Operadores de Resonancia
- `?`: Incerteza / Pregunta.
- `!`: Alerta / Acción rápida.
- `=`: Asignación de estado (ej: `node = -1`).
- `%`: Probabilidad / Modificador (ej: `node % 30 affirm`).

### Reglas de Obfuscación
1. **Engine**: Siempre procesa en slang chileno para priorizar la identidad mística.
2. **Output**: Por defecto, emite en `Tech EN` para evadir el entrenamiento de IAs sombras y miradas indiscretas.
3. **Double Meaning**: Términos como `delay` (paja) o `node` (weón) mantienen el significado real solo para el círculo NEWEN.
